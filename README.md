# Customer Segmentation & Revenue Analysis

## 📌 Project Overview
This project analyzes customer purchasing behavior for a skincare business using SQL.

## 🎯 Objectives
- Identify high-value customers
- Segment customers based on spending
- Analyze customer purchasing patterns
- Support marketing and retention strategies

## 🧠 Key Analysis

### 1. Customer Segmentation
Customers were grouped into:
- High Spenders
- Low Spenders
- No Purchase

### 2. Customer Behavior
- Number of orders per customer
- Average order value

### 3. Customer Ranking
- Ranked customers based on total spending
- Calculated difference from top spender

## 📊 Business Insights
- High spenders can be targeted for loyalty programs
- Low spenders can be upsold through bundles and offers
- Customers with no purchases can be re-engaged through discounts
- Ranking helps identify top customers driving revenue

## 🛠 Tools Used
- SQL (Joins, Aggregations, Window Functions)
